# vi

vi是一种计算机文本编辑器，并且是一种模式编辑器。不同的按钮和键击可以更改不同的“模式”。

vi：维基百科

# vim

Vim是从vi发展出来的一个文本编辑器。其代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。和Emacs并列成为类Unix系统用户最喜欢的编辑器。

vim：维基百科
