# 组管理

  • 查看文件的所有者
  • 修改文件所有者
  • 组的创建
  • 修改文件所在组
  • 其它组
  • 改变用户所在组

# 查看文件的所有者

ls -ahl
1

# 修改文件所有者

chown 用户名 文件名
1

# 组的创建

groupadd 组名
1

# 修改文件所在组

chgrp 组名 文件名
1

# 其它组

除文件的所有者和所在组的用户外，系统的其他用户都是文件的其他组。

# 改变用户所在组

usermod -g 组名 用户名
usermod -d 目录名 用户名 （改变该用户登录的初始目录）
